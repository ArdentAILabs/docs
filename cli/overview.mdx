```bash
npm install -g ardent-cli
```

Run `ardent` with no arguments for onboarding hints, or `ardent --help` for the full command list.

Global options: `--help`, `--version`.

Use the sections in this sidebar for auth, projects, connectors, branches, connector settings, and team commands.
