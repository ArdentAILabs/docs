description: "Install the ardent CLI and browse command groups"
---

```bash
npm install -g ardent-cli
```

Run `ardent` with no arguments for onboarding hints.

To print the full command list in your terminal, run:

```bash
ardent --help
```

That is the same as asking the CLI for help at the top level; you can also use `--help` on any subcommand (for example `ardent branch --help`).

Global options: `--help`, `--version`.
